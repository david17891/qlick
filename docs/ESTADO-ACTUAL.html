Capa 1

Funciona hoy

Todo lo que ves acá corre end-to-end en el panel. Datos reales en la base de datos, flujos verificados.

📚

Catálogo de cursos

5 cursos publicados, con páginas de detalle individuales. Mix de gratuitos, pago único y freemium.

→ /cursos

🎓

Inscripción y acceso

El alumno se inscribe, ve sus cursos inscritos, su progreso lección por lección, y accede al reproductor.

→ /dashboard · /aprender/[lesson]

🔐

Login de un click

Google OAuth como método principal: un solo botón "Entrar con Google" y el usuario está adentro. Magic link por correo queda como segunda opción.

→ /login

📅

Admin de eventos

CRUD completo de eventos con pipeline kanban 5 columnas, métricas de conversión, importar Excel, clonar eventos y deshacer archivar.

→ /admin/eventos

👥

CRM con agente IA

Pipeline kanban de leads, calendario de citas, historial de contactos y 10 intents del agente IA que sugieren la próxima acción comercial.

→ /admin (tab CRM)

📊

Audit log de admin

Cada cambio que un admin hace queda registrado: qué cambió, antes, después, quién y cuándo. Filtros por admin, tipo, fecha.

→ /admin/system/audit-log

📲

WhatsApp pre-armado

Mensajes tipo click-to-chat con el texto ya escrito. El admin hace click y se abre WhatsApp listo para enviar. Broadcast a todos los confirmados con un click.

→ integrado en /admin/eventos

📥

Importar Excel

Subís un Excel con confirmados/asistentes/encuestas y el sistema detecta los headers, muestra preview y sube en lote.

→ /admin/eventos/[id]/import

📝

Formularios con consent

Captura de leads desde landing pages con consentimiento explícito para contacto comercial. Cumple normativa de privacidad.

→ /contacto · /eventos/[slug]

⚡

Capa 2

Casi listo

El código está terminado. Solo falta una decisión o un setup corto (crear cuenta, pegar API key) para activarlo.

📧

Emails automáticos

Cuando entra una encuesta con consentimiento, el sistema manda email al admin avisando del nuevo lead. Falta: crear cuenta en Resend y verificar el dominio. 30 minutos con la guía.

🔒

Aviso de privacidad

El documento legal está redactado y publicado en /privacidad. Falta validación con abogado para uso formal. Una sesión con tu abogado lo deja blindado.

🎬

Videos propios

Hoy los cursos muestran videos de YouTube no-listados (no son videos reales). Falta: producir el contenido y subirlo a Cloudflare Stream o Mux. Cuando estén los videos, se reemplazan en una tarde.

🧪

Modo demo

La app puede correr con datos sintéticos (nombres fake, emails de prueba) sin tocar la base de datos real. Útil para mostrar a socios antes de salir al público. Ya está activo por default.

🌐

Dominio público

El deploy en Vercel está listo (build verde, todas las páginas se generan). Falta: comprar dominio + apuntar DNS a Vercel. Decisión de dominio primero.

🚀

Lo que viene

Próximamente, cuando se decida

Las conexiones ya están preparadas. Cada una se activa con una decisión de negocio y un setup corto.

💳

Cobro real con MercadoPago

Los stubs de pago ya están escritos. Falta elegir proveedor (MercadoPago, Stripe, Conekta) y conectar.

🤖

WhatsApp automatizado

Los stubs de Meta Cloud API ya están listos. Cuando se active: mensajes automáticos a leads, recordatorios de pago, respuestas del agente IA. Requiere aprobación de Meta (~2 semanas).

🧾

Facturación CFDI

Cuando haya cobros reales, se pueden emitir facturas electrónicas (requeridas por ley en México para B2B). Conekta las emite directo; con otros se integra un PAC.

📈

Reportes y analytics

Dashboard con embudo de conversión, revenue mensual, cursos más vendidos, leads por fuente. Se conecta con herramientas como Plausible o Mux Data.

🔑

Passkeys (login con huella)

Login con huella dactilar o Face ID: el usuario toca el sensor y entra. Cero fricción, máxima seguridad. Es lo que usa Apple Pay. 2-3 días de implementación.

🌎

Multi-idioma

Activar inglés para vender cursos al mercado hispano en USA o a cualquier mercado angloparlante. El código está preparado para multi-lenguaje desde el día 1.

🏢

Marca blanca / multi-tenant

Cuando el modelo esté validado, se puede ofrecer la plataforma a otras agencias o marcas con su propio logo, dominio y colores. La arquitectura ya lo soporta sin reescritura.
